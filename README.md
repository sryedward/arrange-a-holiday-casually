# arrange-a-holiday-casually
## 🎯 스마트 랜덤 선택 도구

### 📌 프로젝트 소개

HTML / CSS / JavaScript 기반으로 제작된 웹 앱으로, 여러 선택지 중 하나를 **랜덤으로 선택**해주는 도구입니다.
사용자는 옵션을 입력하면, 회전판 애니메이션을 통해 결과를 확인할 수 있습니다.

### ✨ 주요 기능

* 🎡 회전판 (Canvas)
* 🎯 결과 포인터 표시
* ⏳ 감속 애니메이션
* 🔊 사운드 효과
* 🌙 다크 모드
* 🌍 다국어 지원
* 💾 기록 저장 (LocalStorage)

### 🛠 기술 스택

HTML5 / CSS3 / JavaScript / Canvas API / LocalStorage

## 🎯 智能随机决策工具

### 📌 项目简介

基于 HTML / CSS / JavaScript 开发的网页应用，用于在多个选项中进行**随机选择**。
输入选项后，通过动画转盘自动生成结果。

### ✨ 功能特点

* 🎡 动态转盘
* 🎯 指针结果
* ⏳ 减速动画
* 🔊 音效
* 🌙 夜间模式
* 🌍 多语言
* 💾 本地记录

### 🛠 技术栈

HTML5 / CSS3 / JavaScript / Canvas API / LocalStorage
